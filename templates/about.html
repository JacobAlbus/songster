DebateWatch is a website designed to make national tournament records for high school debaters more accessible. Created by 4 friends at Bridgewater-Raritan High School, the website was born out of what the four founders felt was an essential need for the debate community that had not yet been filled. The backend is programmed in Python, featuring DebateWatch’s proprietary algorithm which enables it to scrape locally stored tournament data and bring it to users' fingertips within seconds. The Python application is deployed via pythonanywhere.com and uses Flask, a web application framework, to facilitate interaction between the frontend and backend. The algorithm itself efficiently analyzes tabroom.com, a website that stores raw data for debaters. It is able to output all necessary information about a singular debater or partnership, acting as a hub of historical records.


With over 20,000 hits in just one month, the site has quickly gained traction within the debate sphere. The site includes records for both Public Forum and Lincoln Douglas debate, ensuring that competitors from all categories are represented. The site also features the premier bid list for Public Forum debate, with timely updates and a thorough record of Gold, Silver, and Single Silver bid teams. It is for this reason that users have called DebateWatch the “total package” when it comes to accessing data from debate tournaments.
